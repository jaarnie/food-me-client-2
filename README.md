# Food Me

Little restaurant searching app using the Zomato API.


### Installation


Client: 
```sh
$ git clone git@github.com:jaarnie/food-me-client-2.git
$ cd food-me-client-2
$ npm install
$ npm start
```

Server:
```sh
$ git clone git@github.com:jaarnie/food-me-server.git
$ cd food-me-serve
$ bundle install
$ rails s
```
